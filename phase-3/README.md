# Course Companion FTE

A full-stack educational Learning Management System (LMS) that operates as a digital tutor 168 hours/week.

## Features

### Free Features
- **User Authentication** - Email/Password, Google, Facebook login
- **Course Browsing** - Browse and enroll in courses
- **Progress Tracking** - Track your learning journey
- **Learning Paths** - Structured learning journeys
- **Quizzes** - Test your knowledge
- **Responsive Design** - Works on mobile, tablet, and desktop

### Premium Features (AI-Powered)
- **Adaptive Learning Paths** - Personalized recommendations
- **LLM-Graded Assessments** - Detailed AI feedback
- **Cross-Chapter Synthesis** - Connect concepts across chapters
- **Advanced Analytics** - Deep insights into your progress

## Demo

This is a demo version with:
- Mock authentication (any email/password works)
- Social login simulation (Google/Facebook)
- Sample course data

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Charts:** Chart.js
- **Icons:** Heroicons

## Getting Started

1. Click "Get Started" or "Login"
2. Use any email and password (min 6 characters) OR
3. Click Google/Facebook for instant demo access
4. Explore the dashboard, courses, and features!
