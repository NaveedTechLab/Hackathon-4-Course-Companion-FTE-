# Frontend Developer Skill

This skill provides expertise in developing Next.js/React web applications for Phase 3 and integrating with the OpenAI Apps SDK for Phase 1. Focuses on responsive LMS dashboards and conversational UI integration.

## Components

### SKILL.md
Core instructions and patterns for frontend development with focus on:
- Next.js/React development best practices
- OpenAI Apps SDK integration for conversational interfaces
- Responsive LMS dashboard development
- Modern UI/UX patterns and component architecture
- TypeScript integration and accessibility

### References
- `nextjs-development.md` - Comprehensive Next.js development guide with App Router, styling, and optimization
- `openai-integration.md` - Complete OpenAI SDK integration with streaming, error handling, and educational context
- `lms-dashboard.md` - LMS dashboard development with components, data visualization, and responsive design

### Scripts
- `create-component.js` - Automated component generator with options for styles, tests, and stories
- `setup-nextjs-project.js` - Complete Next.js project setup with recommended configuration for LMS applications

### Assets
- `component-templates.md` - Standardized templates for common UI components used in educational applications

## Usage

Use this skill when developing:
- Next.js/React web applications with modern UI/UX patterns
- OpenAI Apps SDK integrations for conversational interfaces
- Responsive Learning Management System (LMS) dashboards
- Educational technology applications with rich interactivity
- Conversational UI components for student engagement

## Quick Start

### For Component Creation:
```bash
node scripts/create-component.js MyComponent --with-styles --with-tests
```

### For Project Setup:
```bash
node scripts/setup-nextjs-project.js my-lms-app
```

### For OpenAI Integration:
See `references/openai-integration.md` for complete implementation guides
