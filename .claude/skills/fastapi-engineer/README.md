# FastAPI Engineer Skill

This skill provides expertise in developing deterministic FastAPI backends for content delivery, navigation, and rule-based quiz grading systems. The skill emphasizes Zero-Backend-LLM principles, ensuring no LLM calls are made in core API routes.

## Components

### SKILL.md
Core instructions and patterns for FastAPI development with focus on:
- Content delivery systems
- Navigation features
- Rule-based quiz grading
- Zero-Backend-LLM architecture

### References
- `project-structure.md` - Recommended directory structure and configuration patterns
- `quiz-system.md` - Complete implementation of quiz grading system with models, schemas, and routes
- `content-delivery.md` - Content delivery system with file handling and streaming

### Scripts
- `init_fastapi_project.py` - Script to initialize a new FastAPI project with recommended structure
- `test_setup.py` - Script to verify FastAPI project setup and dependencies

### Assets
- `route-template.md` - Template for creating new API routes with proper structure and patterns

## Usage

Use this skill when developing FastAPI applications that require:
- Content delivery systems for course materials
- Navigation features for course progression
- Rule-based quiz grading systems
- Deterministic backend operations without LLM dependencies
- RESTful API development with proper authentication and authorization
